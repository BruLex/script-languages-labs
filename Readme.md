Intall dependencies:
```bash
$ npm install
```

To run master do next:
```bash
$ npm run start:master
```

To run just worker do next:
```bash
$ npm run start:worker
```

To build container use ``app-control.sh`` script.
```bash
Usage: ./app-control COMMAND
Options:
        --build     Build app
        --deploy    Start app
        --stop      Stop app
```
